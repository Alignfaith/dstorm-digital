Featured Project

Covenant CRM — Inspect what you expect

A customer relationship management platform built for real estate professionals — and adaptable to any relationship-driven business. Covenant CRM directly increases productivity by managing opportunities, contacts, leads, and follow-up in a single focused workflow.

Built entirely by DStorm Digital from concept to production. The platform includes a Next.js 15 web app with Clerk authentication, AI-powered insight generation via Anthropic Claude Haiku 4.5, Prisma ORM, opportunity pipeline tracking, contact and lead management, and structured follow-up workflows.

Next.js 15 TypeScript Clerk Prisma PostgreSQL Claude AI Stripe Vercel
Join the waitlist — covenantcrm.com →
🌐

Web Platform

Full-stack Next.js 15 app with TypeScript, multi-tenant Clerk authentication, server actions, and Tailwind CSS. Deployed on Vercel with edge middleware for access gating.

🤖

AI Insight Engine

Nightly cron job evaluates each contact against three behavioral rules (re-engagement, pre-approval expiring, anniversary) and generates agent-voiced first-draft messages using Anthropic Claude Haiku 4.5. Includes retry logic, dedupe, and manual edit preservation.

🔒

Multi-User Privacy

Per-rep data isolation across contacts, deals, tasks, and insights. Members see only their own records; admins can scope access by user. Backed by row-level ownership fields and a centralized access helper.

🛡️

Admin Tools

Closed beta approval queue, tier management (Standard, Brokerage, Lifetime Free, Internal), audit logging, organization suspension, and platform admin impersonation for support workflows.
