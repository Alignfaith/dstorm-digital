Featured Project

ALIGN — Faith-Centered Relationship Platform

A full-stack relationship readiness platform built on the Six Pillars framework from Thomas Marks' book Relationship Fitness. ALIGN helps Christian singles grow spiritually, mentally, financially, physically, and in character before pursuing a relationship — then algorithmically matches them based on pillar alignment scores.

Built entirely by DStorm Digital from concept to production. The platform includes a Next.js web app with Stripe subscription billing, Supabase PostgreSQL database, a custom six-pillar matching engine, photo moderation, real-time messaging, admin dashboard, and a companion React Native iOS app.

Next.js 14 TypeScript Supabase PostgreSQL Stripe React Native iOS App Vercel Prisma ORM NextAuth.js
View Live — app.alignfaith.com →
🌐

Web Platform

Full-stack Next.js application with user authentication, profile onboarding, Six Pillar assessment, matching engine, real-time messaging, and Stripe subscription billing.

📊

Matching Engine

Custom algorithm that calculates alignment scores across all six pillars, applies weighted scoring, and produces tiered match results (Excellent, Strong, Moderate).

🍎

iOS App

React Native companion app for iOS with Supabase auth, assessment submission, pillar score tracking, alignments tab, and photo upload flow.

🛡️

Admin Dashboard

Full internal admin panel with user management, photo moderation queue, subscription analytics, audit log, and content management for CMS pages.
